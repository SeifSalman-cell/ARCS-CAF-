ARCS CAFÉ — MISE À JOUR SUR GITHUB

Le dossier contient :
- index.html
- css/style.css
- images/

ÉTAPES
1. Décompressez le fichier ZIP.
2. Ouvrez votre dépôt GitHub ARCS-CAF-.
3. Supprimez l’ancien index.html et l’ancien dossier css.
4. Supprimez ou remplacez les anciennes images du dossier images.
5. Cliquez sur « Add file », puis « Upload files ».
6. Déposez index.html, le dossier css et le dossier images à la racine.
7. Conservez votre fichier CNAME.
8. Cliquez sur « Commit changes ».

Si le dépôt est connecté à Vercel, la nouvelle version sera publiée
automatiquement après le commit.
