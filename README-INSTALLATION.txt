ARCS CAFÉ — INSTALLATION SUR GITHUB ET VERCEL

CONTENU DU DOSSIER
- index.html : la page complète du site
- css/style.css : le style vert et doré
- images/ : les quatre photos utilisées sur le site

MISE À JOUR DE VOTRE DÉPÔT GITHUB
1. Décompressez le fichier ZIP.
2. Ouvrez votre dépôt GitHub ARCS-CAF-.
3. Supprimez l’ancien index.html et l’ancien dossier css.
4. Cliquez sur « Add file », puis « Upload files ».
5. Déposez index.html, le dossier css et le dossier images à la racine du dépôt.
6. Conservez votre fichier CNAME si votre domaine est déjà configuré avec GitHub Pages.
7. Cliquez sur « Commit changes ».

PUBLICATION SUR VERCEL
Si le dépôt est déjà connecté à Vercel, une nouvelle publication sera lancée
automatiquement après le commit.

Pour une première connexion :
1. Dans Vercel, cliquez sur « Add New » puis « Project ».
2. Importez le dépôt GitHub ARCS-CAF-.
3. Choisissez « Other » comme Framework Preset.
4. Laissez Build Command vide.
5. Indiquez « . » comme Output Directory si Vercel le demande.
6. Cliquez sur « Deploy ».

IMPORTANT
Le menu détaillé et les prix ne sont pas encore ajoutés. Ils pourront être
insérés plus tard dans index.html sans changer le style général du site.
