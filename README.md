## Mettre le site en ligne avec GitHub Pages

1. Crée un nouveau repository sur GitHub (par ex. `arcs-cafe`).
2. Mets tous les fichiers de ce dossier dedans (via l'interface web "Add file → Upload files", ou en ligne de commande avec `git`).
3. Dans le repository : **Settings → Pages**.
4. Dans "Branch", choisis `main` et le dossier `/ (root)`, puis **Save**.
5. Après une minute ou deux, ton site sera visible à l'adresse indiquée en haut de cette page (un lien du type `https://ton-pseudo.github.io/arcs-cafe/`).
